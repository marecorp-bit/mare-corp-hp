## デモページの仕様（demo.html）

- 既存のギフトページ（index.html）のデザイン・スタイルをベースに新規作成
- コード入力画面をスキップして、いきなり商品選択画面を表示する
- 画面上部に「デモ版」バナーを常時表示する
- 発注ボタンを押したら「このページはデモです。実際の発注はできません」とポップアップ表示
- 商品データはGAS APIを使わず、架空のサンプル商品をHTMLにハードコードして表示する
- 「こんなギフトページが作れます」という紹介・提案用途のページ

## 開発環境

- フロント：HTML / CSS / JavaScript
- ホスティング：Cloudflare Pages（無料・商用利用OK）
- コード管理：GitHub（新規リポジトリを作成する。例：`mare-corp-hp`）
- 開発ツール：Claude Code

### Claude Codeの起動手順
1. デスクトップに新しいフォルダを作成（例：`mare-corp-hp`）
2. コマンドプロンプトで `cd OneDrive\デスクトップ\mare-corp-hp`
3. `claude` で起動
4. 「デプロイして」と伝えるだけでGitHub経由でCloudflare Pagesに自動反映
